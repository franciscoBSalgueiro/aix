# Aix - `aixchess` DuckDB extension (En Croissant fork)

Aix enables efficient storage and querying of large chess game collections. This fork was made to enable support for variations and custom starting positions in order to be usable by [En Croissant](https://github.com/franciscoBSalgueiro/en-croissant) (See [issue #4](https://github.com/thomas-daniels/aix/issues/4)). Thank you Thomas for creating Aix!
